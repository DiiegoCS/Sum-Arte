# Sum-Arte
Sum-Arte es un sistema de gestión de recursos financieros para iniciativas culturales, en formato de Plataforma Web especializa.

La solución digitaliza y automatiza el ciclo completo de los proyectos adjudicados, incorporando funciones de control financiero, gestión documental, trazabilidad de evidencias y reportería inteligente. Con ello, Sum-Arte busca optimizar la rendición de cuentas, fortalecer la transparencia y mejorar la sostenibilidad de las organizaciones, posicionándose como una herramienta estratégica para la innovación en la gestión cultural.
